# OP3.26
Organisasi Pelajar

Situs demo sederhana untuk "Organisasi Pelajar" (OP3.26) dibuat dengan Tailwind CSS (CDN) dan GSAP untuk animasi.

Quick start

1. Buka `index.html` di peramban modern.
2. File ini menggunakan Tailwind via CDN, jadi tidak perlu instalasi.

Notes

- Untuk produksi, pertimbangkan membangun Tailwind secara lokal dan mengoptimalkan aset.
- GSAP dipakai untuk animasi masuk dan scroll (via CDN). Anda bisa mengganti atau menambah plugin GSAP.

Jika ingin, saya bisa:

- Tambahkan halaman admin sederhana.
- Tambahkan formulir kerja (server) atau integrasi Google Forms.
- Mengonversi proyek ke stack Node.js + PostCSS untuk membangun Tailwind.
